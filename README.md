# Paprika_RedesFinalP

Setup

Descargar imagenes-docker.tar

Correr el siguiente comando 

```
docker load -i imagenes-docker.tar
```

Una vez subidas las imágenes se puede importar el proyecto en GNS3.

Si la importación no funciona porque una imagen hace falta, se recomienda cambiar el nombre de algún cliente al de la imagen faltante para poder importarlo sin inconvenientes.
